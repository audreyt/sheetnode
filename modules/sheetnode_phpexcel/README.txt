SHEETNODE_PHPEXCEL v7.x

A sub-module to extend the functionality of the SHEETNODE module to import 

and export EXCEL spreadsheets using the PhpExcel project



INSTALLATION

See documentation at drupal.org/project/sheetnode.



STATUS

This has been modified for Drupal v7.x so that it has successfully 
imported 
.xlsx spreadsheets. Other functionality has not been tested, however.
