SHEETNODE_TEXT v7.x

A sub-module to extend the functionality of the SHEETNODE module to provide 
text table import/export 
support for Sheetnode.



INSTALLATION

See documentation at drupal.org/project/sheetnode.



STATUS

This is effectively the v6.x version except that the .info file was changed
so that it will not show 
an error when installed under 7.x.
