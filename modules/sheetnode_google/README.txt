SHEETNODE_GOOGLE v7.x

A sub-module to extend the functionality of the SHEETNODE module to import 

and export spreadsheets from Google. 



INSTALLATION

See documentation at drupal.org/project/sheetnode.



STATUS

This is effectively the v6.x version except that the path for configuration 

was modified so it appears correctly under the Configure >> Content section 

in Drupal 7.x.
